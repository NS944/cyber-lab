# Day 1 — Linux Lab Setup & Basics

## What I did (EN)
- Installed Ubuntu Server on VirtualBox
- Created a virtual machine and configured resources (RAM, CPU, disk)
- Fixed boot issues and mounted ISO correctly
- Created user account and logged into system
- Installed OpenSSH server and connected via SSH from Windows
- Updated system using apt
- Learned basic Linux commands (pwd, ls, cd, mkdir, touch)
- Created and edited files using nano
- Ran my first Python script

## What I learned (EN)
- How a Linux server works in a virtual environment
- How to navigate the filesystem using terminal
- How SSH allows remote control of another machine
- How to create, edit, and read files in Linux
- Basic workflow of running Python scripts in Linux
